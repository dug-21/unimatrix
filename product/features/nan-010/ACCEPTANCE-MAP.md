| AC-10 | Fail condition distinguishes "Diversity targets not met" from "Diversity targets met, but ranking floor breached"; the two failure modes are reported separately in rendered output | test | `test_distribution_gate_distinct_failure_modes` in `eval/report/tests_distribution_gate.rs`; two fixtures — one where CC@k fails, one where CC@k/ICD pass but MRR floor fails; assert distinct rendered messages | PENDING |
| AC-11 | When `profile-meta.json` is absent from the results directory (pre-nan-010 result sets), `eval report` treats all profiles as `distribution_change = false` and renders Section 5 as "Zero-Regression Check"; no error is emitted | test | `test_report_without_profile_meta_json` in `eval/report/tests_distribution_gate.rs`; point report at results dir with no `profile-meta.json`; assert "Zero-Regression Check" heading and exit 0 | PENDING |
