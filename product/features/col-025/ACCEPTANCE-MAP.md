## Additional Non-Negotiable Test Coverage (from RISK-TEST-STRATEGY)

| Scenario | Risk | Verification Detail |
|----------|------|---------------------|
| UTF-8 char-boundary truncation at UDS 4096-byte boundary | R-07 | Unit test: supply goal whose 4096th byte falls mid-character (e.g., 3-byte CJK spanning bytes 4095–4097); assert UDS handler does not panic; assert stored goal is valid UTF-8 with length ≤ 4096 bytes. Uses `UDS_MAX_GOAL_BYTES = 4096` constant. |
| Full column-value assertion on `insert_cycle_event` round-trip | R-08 | Integration test: write `cycle_start` event with known goal; read back full row; assert EACH column (`event_type`, `phase`, `outcome`, `next_phase`, `goal`) contains its expected value — not just `goal` in isolation. |
| `SubagentStart` with empty `prompt_snippet` AND `current_goal = None` | R-04 | Unit test: assert fallback to `RecordEvent` or topic (no goal-derived query issued). |
| `insert_cycle_event` call site audit | R-01 | Pre-delivery: grep `crates/` for `insert_cycle_event`; confirm exactly one call site in `listener.rs` before modifying the function signature. |
