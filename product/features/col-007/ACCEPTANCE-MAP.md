| AC-03 | Search pipeline produces equivalent results via MCP and UDS | integration test | Populate knowledge base with 5+ entries. | AC-04 | Formatted stdout includes title, category, confidence, content | unit test | Call `format_injection()` with known `EntryPayload` entries. | AC-05 | Token budget enforced (1400 bytes) | unit test | Call `format_injection()` with entries whose combined content exceeds 1400 bytes. Verify `output.len() <= MAX_INJECTION_BYTES`. Verify entry omitted entirely if remaining budget < 100 bytes. | R-03 (byte budget overflow) | injection-format | PENDING |
| AC-06 | Co-access pairs generated with session dedup | integration test | Send ContextSearch returning 3 entries for session "s1". Verify co-access pairs recorded via `store.record_co_access()`. Verify new co-access pairs recorded. Send SessionClose for "s1". Verify dedup set cleared. | R-06 (memory leak) | uds-dispatch, session-warming | PENDING |
| AC-07 | SessionStart pre-warms ONNX model | integration test | Send `SessionRegister` via UDS. Wait for Ack. Send `ContextSearch` immediately after. Verify ContextSearch returns `Entries` with results (not empty EmbedNotReady). | AC-08 | Graceful degradation when server unavailable | unit test | Run `unimatrix-server hook UserPromptSubmit` with synthetic stdin but no running server (no socket file). Verify exit code 0. | AC-09 | Silent skip on empty/low-quality results | unit test + integration test | Unit test: `format_injection()` with empty entries slice returns `None`. Unit test: `format_injection()` with entries where all similarity < 0.5 returns `None` (entries should not reach format_injection if filtered server-side, but verify graceful handling). Integration test: ContextSearch with query unrelated to knowledge base returns `Entries` with empty `items`. | R-04 (threshold suppression) | injection-format, uds-dispatch | PENDING |
| AC-10 | HookInput.prompt field works correctly | unit test | Deserialize JSON with `prompt` field present: verify `prompt` is `Some("text")`. Deserialize JSON with both `prompt` and unknown fields: verify `prompt` in named field, others in `extra`. | R-07 (flatten conflict) | hook-handler | PENDING |
| AC-11 | Existing MCP integration tests pass | shell | `cargo test --workspace` passes. All existing 1025 unit + 174 integration tests pass without modification. Zero new failures. | AC-12 | Hot-path latency under 50ms | benchmark test | Start server with populated knowledge base. Send SessionRegister (trigger warming). Individual iteration timings logged for analysis. | R-08 (timeout), R-11 (spawn_blocking) | All components (end-to-end) | PENDING |
